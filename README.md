# **SameName TVShowSrt**

**Tools used:**
* Electron
* HTML
* CSS
* JavaScript / Node.js

**Description**\
Tired of renaming video and srt files one by one?\
Enter the show Name, select the folder, next click Rename Files and you are done.

The video and srt file have to have the Season and Episode number:\
asdf.S01E01.mkv\
asdf.S01E01.srt
